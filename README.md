# Next Hire — Freelance Recruitment Platform

CSC456 Project 3 — Full stack Node.js + Express + MongoDB + EJS app with role-based auth (Admin, Client, Freelancer).

## Live Demo
https://csc456-project03.onrender.com

## GitHub
https://github.com/georgioai/csc456-project03

## Tech Stack
Node.js, Express, MongoDB Atlas, EJS, bcrypt, JWT, Render, GitHub Actions

## Run Locally
1. `npm install`
2. Add `.env` with MONGO_URI, JWT_SECRET, SESSION_SECRET
3. `node server.js`
